    Purpose of each Stage

    Project Phase 1

    In the first phase of the project, we will look at the CAN system. There will be 3 floor nodes connected to the CAN system that will call for the elevator. There will also be an elevator node that will tell the system which floor to go. The elevator will consist of a button for each floor and two more buttons to open and close the elevator door.

    Project Phase 2

    In the second phase of the project, we will look at the GUI and Database. The GUI will provide a user control of the elevator from a remote site that can access the elevator's floors and log files of data being transmitted through the CAN system.

    Project Phase 3

    In the third phase of the project, we will integrate the GUI/Database will the running CAN system.
